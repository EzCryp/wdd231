Site Name

Gadget Cycle

This name was selected because it clearly represents the website's focus on gadgets buying and selling cycle.

Plan Site URL: www.gadgetcycle.com

Site Logo

The logo will be a simple design from one of Gadget Cycle Material.

Site Purpose

The site purpose is to provide a platform for users to buy and sell gadgets easily.

Site Tagline:"Turn Your Old Tech into Cash or Find Your Next Gadget"

Scenarios

  • What is everyone's current Gadget Cycle?
  • Where can they get new and upgraded gadgets?
  • Where can we sell our old yet useful gadgets?

Color Schema

  • Dark Blue: Used for headings, primary text, accents, and primary elements.
  • Light Green: Used for buttons, link hover effects, and secondary elements.
  • Alice Blue: Used for background colors.
  • Medium Turquoise: optional accent color.
  • Light Gray: Used for muted text color.
  • White: Used for text color option 2.

Typography

  • Roboto:Used for headings and titles with larger weight. Used for body text and paragraphs with smaller weight and additional italic.

Pages

Order

The Buy / Sell Gadget page will allow users to buy and sell gadgets within the Gadget Cycle community.
